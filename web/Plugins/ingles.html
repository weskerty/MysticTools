⚙️ Variables Extra

Se puede Cambiar los ModelosIA por otros Disponibles en Groq ↗️. Por Defecto Utiliza llama-3.1-70b-versatile.

Motor de Voz

.setvar global.vozgroq = modelo

Motor de Texto

.setvar global.textogroq = modelo

Personalidad

.setvar global.personalidadgroq = como debe ser

Cada cambio de estos Reiniciara el Bot. Tambien puedes Ajustarlos Directamente en el Plugin para Evitar el Reinicio.
